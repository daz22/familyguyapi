Family Guy API (GET only)

A family guy API. I wanted to learn about more about APIs so I decided to build this repo and learn. 
This repo publishes static JSON under /api/v1.
GitHub Pages serves the JSON and a simple docs site.


Paths:
- /api/v1/characters.json
- /api/v1/characters/{id}.json
- /api/v1/quotes.json
- /api/v1/episodes.json
